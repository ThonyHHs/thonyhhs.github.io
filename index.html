Alguns projetos que já fiz

Snake Game

Este projeto, desenvolvido em C, implementa o clássico jogo da cobra (Snake) jogável diretamente no terminal. O jogo utiliza controles básicos para movimentar o jogador, comer frutas e crescer, enquanto o jogador evita colidir com a própria cauda.

Pykemon

Este projeto, desenvolvido em python, é um jogo de aventura inspirado em Pokémon, jogado no terminal. Os jogadores exploram biomas, capturam pokémons e visualizam seus itens e pokédex. O código usa módulos e funções para organizar as mecânicas do jogo.

Sistema Acadêmico

Este projeto em Python implementa um sistema de gerenciamento acadêmico que usa arquivos JSON para armazenar dados e permite realizar operações de CRUD (Criar, Ler, Atualizar e Excluir) para alunos, professores, disciplinas, turmas e matrículas.

Miner

Este projeto implementa o clássico jogo de Campo Minado, jogável diretamente no navegador. Desenvolvido em JavaScript, HTML e CSS, o jogo permite ao usuário selecionar a dificuldade, colocar bandeiras, revelar células e evitar bombas em um grid dinâmico.
